# portfolio

Static portfolio site for [www.jflab.cz](https://www.jflab.cz).


## Local preview

You can open `index.html` directly in a browser, or run a tiny local web server:

```bash
cd portfolio
python3 -m http.server 8000
```

Then open <http://127.0.0.1:8000>.

## Project structure

```text
portfolio/
├── index.html        # Main page content
├── static/
│   ├── styles.css    # Site styling
│   └── fa/           # Local Font Awesome assets
└── README.md
```
